### To Local JSON-SERVER DataBase :

To install the [JSON-SERVER](https://github.com/typicode/json-server) locally

```
npm install -g json-server
```

To Run json-server

```
npx json-server --watch db.json
```

It will run on baseURI Port number:

```
http://localhost:3000
```
